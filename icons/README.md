## Quick Generation

1. **Online Tools:**
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator
   - https://favicon.io/favicon-generator/

2. **Create a 512x512 master image:**
   - Use your logo or design
   - Background: #6366f1 (primary color)
   - Icon/emoji: White or contrasting color

3. **Generate all sizes** using one of the tools above

4. **Place all files** in this `icons/` folder
